# Changelog

## [0.1.0] — 2026-03-31

### Added
- Filesystem-wide markdown scanner with smart ignore rules
- WYSIWYG editor powered by Milkdown with raw markdown toggle
- Full-text search across file names, paths, and content
- Collapsible file tree with per-folder expand/collapse on hover
- Git awareness — detects which files live in git repos
- Daemon mode with start/stop/status/install/uninstall
- Auto-start on boot via systemd (Linux) or launchd (macOS)
- Scan caching for instant restarts
- File watcher with live WebSocket updates
- Auto-refresh when files are modified externally
- Floating selection toolbar (bold, italic, strikethrough, code, link)
- Anchor link navigation within documents
- Scroll-to-top button
- Copy file path on click
- VS Code integration (open file directly)
- Keyboard shortcuts (Ctrl+S save, Ctrl+B bold, Ctrl+I italic)
- URL hash persistence — refresh preserves open file
- Linear-inspired dark theme
- Zero runtime dependencies
